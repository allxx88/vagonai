{{- if and .IsPage (eq .Section "blog") -}} {{- $desc := .Description -}} {{- if not $desc -}} {{- $desc = .Summary -}} {{- end -}} {{- $desc = $desc | plainify | htmlUnescape | truncate 160 -}} {{- $published := (.Date.Format "2006-01-02T15:04:05Z07:00") -}} {{- $modified := $published -}} {{- if .Lastmod -}} {{- $modified = (.Lastmod.Format "2006-01-02T15:04:05Z07:00") -}} {{- end -}} {{- $ld := dict "@context" "https://schema.org" "@type" "Article" "headline" .Title "datePublished" $published "dateModified" $modified "url" .Permalink "mainEntityOfPage" (dict "@type" "WebPage" "@id" .Permalink) -}} {{- if $desc -}} {{- $ld = merge $ld (dict "description" $desc) -}} {{- end -}} {{- if .Params.author -}} {{- $ld = merge $ld (dict "author" (dict "@type" "Person" "name" .Params.author)) -}} {{- else -}} {{- $ld = merge $ld (dict "author" (dict "@type" "Organization" "name" "ПРОВАГОН" "url" .Site.BaseURL)) -}} {{- end -}} {{- $ld = merge $ld (dict "publisher" (dict "@type" "Organization" "name" "ПРОВАГОН" "url" .Site.BaseURL)) -}} {{- if .Params.image -}} {{- $img := (.Params.image | absURL) -}} {{- $ld = merge $ld (dict "image" (slice $img)) -}} {{- else if .Params.images -}} {{- $imgs := apply .Params.images "absURL" "." -}} {{- $ld = merge $ld (dict "image" $imgs) -}} {{- end -}} {{- end -}}
